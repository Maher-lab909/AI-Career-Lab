# Project 01 Summary — CV Parser

## What I Built

I built a Python CV Parser that reads PDF CV files and extracts candidate contact information.

The parser extracts:

- Candidate name
- Email address
- Phone number

It saves the results into:

- JSON
- Excel
- Log file

## Why It Matters

Recruiters usually work with Excel, not JSON.

The Excel output makes the extracted results easier to review, filter, and share.
